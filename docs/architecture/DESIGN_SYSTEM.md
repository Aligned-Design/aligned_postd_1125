# Aligned AI Design System Guidelines for AI Assistants

## Project Architecture (Fusion Starter)

This is a **React SPA with Vite + Express** project following Fusion Starter patterns:
- ✅ **Client-focused SPA** with React Router 6
- ✅ **Minimal Express server** (only for private API keys, DB operations)
- ✅ **TypeScript throughout** with shared types
- ✅ **TailwindCSS 3** for styling
- ✅ **Radix UI** for accessible components
